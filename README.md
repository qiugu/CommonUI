# common-ui

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 项目描述
记录工作中常用的一些原生vue组件，不受UI框架限制，业务组件比较多，如果组件有什么不了解可以联系我

### 组件
#### floating组件
属性|描述|默认
-|-|-
floatingBodyClass|悬浮窗的融资样式|默认为{}
rate|悬浮窗的移动速度|默认为10
interval|悬浮窗的移动间隔时间，单位为毫秒|默认为60
